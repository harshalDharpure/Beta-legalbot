# Beta-legalbot

Multilingual legal NLP + legal dialogue generation (POCSO domain). This repository contains:

- **Generation experiments (Exp1–Exp5)** and evaluation tooling.
- **Hybrid RAG pipeline** (retrieval + rerank + citation prompt + best-of-N).
- **`research_pipeline/`**: a reproducible 3-stage research framework (SFT → multi-objective → DPO) with strict train/val/test protocol.
